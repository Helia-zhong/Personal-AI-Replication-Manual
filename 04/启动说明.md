# 启动说明

## 方法1：双击启动脚本
直接双击 `启动.bat` 文件

## 方法2：命令行启动
打开命令提示符（CMD）或 PowerShell，执行：

```bash
cd f:/项目/crank-rocker-mechanism
python -m streamlit run app.py
```

## 方法3：指定端口启动
如果默认端口被占用，可以指定其他端口：

```bash
python -m streamlit run app.py --server.port 8502
```

## 启动后
- 浏览器会自动打开 http://localhost:8501
- 如果没有自动打开，请手动在浏览器中访问该地址
- 使用左侧滑块调节参数，实时查看仿真结果

## 常见问题

### 问题1：端口被占用
错误信息：`Address already in use`

解决方法：
```bash
python -m streamlit run app.py --server.port 8502
```

### 问题2：浏览器没有自动打开
手动打开浏览器，访问：http://localhost:8501

### 问题3：依赖缺失
运行以下命令重新安装：
```bash
python -m pip install -r requirements.txt
```

## 停止服务
在命令行窗口按 `Ctrl + C`
