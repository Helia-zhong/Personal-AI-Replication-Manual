# 工程级曲柄摇杆机构运动学仿真与轨迹分析系统

## 项目简介

这是一个基于严谨机械原理的曲柄摇杆机构仿真系统，实现了完整的运动学分析和传动性能评估。

## 核心功能

### 1. 格拉晓夫定理校验
- 自动判断输入尺寸能否构成曲柄摇杆机构
- 实时检测机构闭合条件
- 区分双摇杆、双曲柄、曲柄摇杆等机构类型

### 2. 矢量闭合求解
- 基于解析法求解机构位置
- 计算连杆中点P的完整运动轨迹
- 精确求解各杆件角度

### 3. 传动角监控
- 实时计算传动角γ（连杆与摇杆夹角）
- 识别最小传动角位置
- 工程死角预警（γ_min < 40°）

### 4. 可视化分析
- 等比例机构姿态图
- 点P封闭运动轨迹
- 传动角变化曲线

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行方式

```bash
streamlit run app.py
```

## 使用说明

1. 通过左侧边栏的滑块调节四个连杆长度
2. 系统自动校验格拉晓夫条件
3. 如果配置有效，主界面显示机构运动轨迹和传动角分析
4. 如果传动角小于40°，系统会发出工程死角预警

## 技术特点

- **严谨的机械原理计算**：完全基于矢量闭合方程和余弦定理
- **实时约束校验**：不满足格拉晓夫条件时立即阻止渲染
- **等比例可视化**：确保机构图形不变形
- **工程级预警系统**：传动角监控与死角识别

## 默认参数示例

- L₁ (机架): 200 mm
- L₂ (曲柄): 60 mm
- L₃ (连杆): 180 mm
- L₄ (摇杆): 150 mm

此配置满足格拉晓夫条件，可构成标准曲柄摇杆机构。

## 注意事项

- 曲柄长度L₂应为最短杆或次短杆
- 机架长度L₁不应为最长杆
- 传动角建议保持在40°以上以确保良好传动性能
