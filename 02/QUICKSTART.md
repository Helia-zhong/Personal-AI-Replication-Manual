# 快速入门指南

## 🚀 启动应用

### 方法 1: 使用启动脚本 (推荐)

**Windows 用户:**
```bash
run.bat
```
双击 `run.bat` 文件或在命令行中运行

**Linux/Mac 用户:**
```bash
chmod +x run.sh
./run.sh
```

### 方法 2: 使用命令行

```bash
streamlit run app.py
```

或者

```bash
python -m streamlit run app.py
```

应用将自动在浏览器中打开,默认地址: http://localhost:8501

## 📖 使用教程

### 第一步: 手动调参体验

1. 启动应用后,您会看到左侧控制面板和右侧图表区域
2. 在左侧的 "手动 PID 参数" 区域,尝试调整三个滑块:
   - **Kp (比例系数)**: 控制响应速度,过大会导致震荡
   - **Ki (积分系数)**: 消除稳态误差,过大会导致超调
   - **Kd (微分系数)**: 减少超调,提高稳定性

3. 观察右侧 "阶跃响应对比" 图表的实时变化
4. 查看顶部的性能指标:
   - 超调量越低越好 (理想值: 0%)
   - 调节时间越短越好
   - 稳态误差越小越好

**尝试这些参数组合:**
- 快速但有超调: Kp=20, Ki=5, Kd=2
- 平稳但较慢: Kp=5, Ki=0.5, Kd=3
- 默认设置: Kp=10, Ki=1, Kd=1

### 第二步: AI 自动寻优

1. 在左侧边栏找到 "AI 寻优参数" 区域
2. 调整 PSO 算法参数:
   - **粒子数量**: 20-30 (初学者推荐 20)
   - **迭代次数**: 30-50 (初学者推荐 30)
   - **目标转角**: 保持默认 30° 或自定义

3. 点击 "🚀 启动 AI 自动寻优" 按钮
4. 等待 10-30 秒,AI 正在智能搜索最优参数...
5. 寻优完成后:
   - 左侧显示 AI 优化的 PID 参数
   - 右侧绿色曲线显示 AI 优化后的响应
   - 查看 "PSO 算法收敛曲线" 了解优化过程

### 第三步: 对比分析

AI 寻优完成后,您可以:
- 对比蓝色曲线(手动)和绿色曲线(AI)的差异
- 查看性能指标的改进程度 (带绿色/红色箭头)
- 分析 PSO 收敛曲线,了解算法如何一步步找到最优解

## 🔧 故障排除

### 问题 1: 应用无法启动
**解决方案:**
```bash
# 检查依赖是否安装
python -c "import streamlit; print(streamlit.__version__)"

# 重新安装依赖
pip install -r requirements.txt --upgrade
```

### 问题 2: AI 寻优速度慢
**解决方案:**
- 减少粒子数量 (尝试 15)
- 减少迭代次数 (尝试 20)
- 这会稍微降低寻优质量,但显著提升速度

### 问题 3: 图表不显示
**解决方案:**
- 刷新浏览器页面 (F5)
- 清除浏览器缓存
- 尝试使用 Chrome 或 Firefox 浏览器

## 📚 学习资源

### PID 控制理论
- Kp: 比例控制,决定响应速度,过大会震荡
- Ki: 积分控制,消除稳态误差,过大会超调
- Kd: 微分控制,预测未来误差,减少超调

### PSO 算法原理
粒子群优化模拟鸟群觅食行为:
- 每个粒子代表一组 PID 参数
- 粒子记住自己找到的最佳位置
- 粒子互相分享全局最佳位置
- 通过迭代不断接近最优解

### 适应度函数
ITAE (时间乘绝对误差积分):
- 考虑误差大小和持续时间
- 惩罚后期的误差
- 综合评价控制性能
