## 🔍 核心代码结构

### 1. PSO 类 (粒子群优化算法)
```
PSO
├── __init__()          # 初始化粒子位置和速度
├── optimize()          # 执行优化主循环
│   ├── 评估适应度
│   ├── 更新个体最佳
│   ├── 更新全局最佳
│   └── 更新粒子速度和位置
└── 返回最优参数
```

### 2. SteeringSystem 类 (转向系统仿真)
```
SteeringSystem
├── __init__()                    # 初始化系统模型
├── simulate_step_response()      # PID 闭环仿真
│   ├── 状态空间建模
│   ├── PID 控制计算
│   └── 系统响应输出
└── calculate_performance_metrics() # 性能指标计算
    ├── 超调量
    ├── 调节时间
    └── 稳态误差
```

### 3. fitness_function() (适应度函数)
```
fitness = ITAE + 权重×超调量 + 权重×调节时间 + 权重×稳态误差
```

### 4. main() (Streamlit 界面)
```
main()
├── 页面配置和样式
├── 系统初始化
├── 侧边栏控制
│   ├── 手动参数滑块
│   └── AI 寻优按钮
├── 主界面显示
│   ├── 性能指标对比
│   ├── 阶跃响应图表
│   └── PSO 收敛曲线
└── 实时交互更新
```

## 📊 技术参数

### 系统模型
- **类型**: 二阶欠阻尼系统
- **传递函数**: G(s) = 100 / (s² + 10s + 100)
- **自然频率**: ωn = 10 rad/s
- **阻尼比**: ζ = 0.5

### PSO 参数
- **默认粒子数**: 20
- **默认迭代数**: 30
- **惯性权重**: w = 0.7
- **学习因子**: c1 = c2 = 1.5
- **搜索空间**: P∈[0,50], I∈[0,10], D∈[0,10]

### 仿真参数
- **时间步长**: 0.001 秒
- **仿真时长**: 2.0 秒
- **调节时间误差带**: ±2%
