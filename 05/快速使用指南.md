# 快速使用指南

## 最简单的方式（推荐）

### Windows 用户
双击 `启动.bat` 文件，选择选项 1，即可在浏览器中打开项目。

### Mac/Linux 用户
在终端中运行：
```bash
./启动.sh
```
然后选择选项 1。

### 或者直接打开
直接双击 `index.html` 文件，用浏览器打开即可。

---

## 项目特点

✅ **零配置运行** - 无需安装任何依赖，直接打开 HTML 即可
✅ **实时数据** - 每秒更新转速和温度数据
✅ **智能预警** - 温度超过 80°C 自动显示红色闪烁警告
✅ **科技感 UI** - 深色主题，青色高亮，专业大屏风格

---

## 功能演示

1. **转速仪表盘**（左侧）
   - 实时显示 1500-1600 RPM
   - 动态指针跟随数值变化
   - 大号数字显示当前转速

2. **温度折线图**（右侧）
   - 显示过去 60 秒的温度变化
   - 红色虚线标记 80°C 警戒线
   - 鼠标悬停查看具体数值

3. **超温预警**（右上角）
   - 温度 > 80°C 时自动出现
   - 红色背景 + 闪烁动画
   - 警告图标 + 文字提示

4. **状态面板**（底部）
   - 当前转速、温度、运行状态
   - 超温时数值变红

## 如果需要开发

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 修改代码：
   - 编辑 `src/` 目录下的文件
   - 浏览器会自动刷新

## 问题排查

**Q: 打开后页面空白？**
- 检查浏览器控制台是否有错误
- 确保网络连接正常（需要加载 CDN 资源）
- 尝试使用 Chrome 或 Edge 浏览器

**Q: 数据不更新？**
- 刷新页面重试
- 检查浏览器控制台是否有 JavaScript 错误

**Q: 想修改数据范围？**
- 编辑 `index.html` 文件
- 搜索 `Math.random() * 101 + 1500` 修改转速范围
- 搜索 `Math.random() * 35 + 50` 修改温度范围
