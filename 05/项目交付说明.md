## 🚀 快速开始

### 方式一：直接运行（推荐）
```bash
# Windows
双击 启动.bat，选择选项 1

# Mac/Linux
./启动.sh
选择选项 1

# 或直接打开
双击 index.html 文件
```

### 方式二：开发模式
```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问 http://localhost:5173
```

## 📝 使用说明

### 无需安装依赖
`index.html` 使用 CDN 加载所有依赖，可以直接在浏览器中运行，无需任何安装步骤。

### 如需修改代码
1. 编辑 `src/` 目录下的组件文件
2. 运行 `npm run dev` 启动开发服务器
3. 浏览器会自动热重载

### 如需修改数据范围
编辑 `index.html` 或 `src/App.jsx`：
- 转速范围：`Math.random() * 101 + 1500`
- 温度范围：`Math.random() * 35 + 50`
- 警告阈值：`newTemp > 80`
